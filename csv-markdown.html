CSV → Markdown 테이블 변환기

CSV 파일을 Markdown 표로 즉시 변환합니다. GitHub README, 블로그, 노션에 바로 붙여넣기.

📄

CSV 파일을 여기에 놓으세요

.csv .xlsx 파일 | 최대 20MB

💡 사용 팁

  • GitHub README, 노션, GitLab, Obsidian 등 Markdown을 지원하는 모든 플랫폼에서 사용 가능합니다.
  • 열 이름에 파이프(|) 문자가 있으면 자동으로 이스케이프됩니다.
  • 미리보기 탭에서 실제 표 모양을 확인할 수 있습니다.
  • 최대 행 수를 제한하면 큰 CSV에서 일부만 추출할 수 있습니다.

FAQ

어떤 플랫폼에서 사용할 수 있나요?

GitHub, GitLab, Bitbucket README, 노션, Obsidian, Jekyll, Hugo 등 Markdown을 지원하는 모든 플랫폼에서 사용 가능합니다.

열이 많으면 표가 깨지나요?

Markdown 표 자체는 열 수 제한이 없지만, 열이 많을수록 가독성이 떨어질 수 있습니다. 필요 열만 선택하려면 열 선택 도구를 먼저 사용하세요.

파일이 서버에 전송되나요?

아니요. 모든 변환은 브라우저에서만 처리됩니다.
