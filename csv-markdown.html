자주 묻는 질문

생성된 Markdown 표를 GitHub에서 사용할 수 있나요?

네. GitHub, Notion 등 Markdown을 지원하는 곳에서 바로 사용 가능합니다.

열 정렬(:--:, ---:)도 지원되나요?

네. 열별 정렬 방향을 설정할 수 있습니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 개인정보와 민감한 데이터가 외부로 유출될 걱정이 없으며, 기업 보안 환경에서도 안심하고 사용할 수 있습니다.

💡 사용 팁

  • GitHub README, 노션, GitLab, Obsidian 등 Markdown을 지원하는 모든 플랫폼에서 사용 가능합니다.
  • 열 이름에 파이프(|) 문자가 있으면 자동으로 이스케이프됩니다.
  • 미리보기 탭에서 실제 표 모양을 확인할 수 있습니다.
  • 최대 행 수를 제한하면 큰 CSV에서 일부만 추출할 수 있습니다.
