💡 Tips

  • Empty row removal: only removes rows where ALL cells are blank. Rows with some data are kept
  • Empty column removal: only removes columns that are entirely blank including the header
  • Whitespace trim: removes leading/trailing spaces only; spaces in the middle are kept

FAQ

Can I remove duplicate rows at the same time?

For duplicate row removal, please use the dedicated "Remove Duplicates" tool.

Is the header row removed?

No. The first row is treated as a header and is never removed.

Is the original file changed?

No. Your original file is never modified. Only the cleaned result is saved as a new file.
