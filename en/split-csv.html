FAQ

How many rows per file is recommended?

Excel supports up to 1,048,576 rows. Splitting into 1,000–10,000 rows per file is generally recommended.

How do I open the ZIP file?

On Windows, open it directly in File Explorer. On Mac, double-click to extract.

Is the original file kept?

Yes. Your original file is not changed. Only the split result is downloaded.
