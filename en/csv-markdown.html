💡 Tips

  • Works with GitHub, GitLab, Bitbucket, Notion, Obsidian, Jekyll, and any Markdown platform.
  • Pipe characters (|) inside values are automatically escaped.
  • Use the Preview tab to see how your table will look when rendered.
  • Limit rows to extract a subset from a large CSV file.

FAQ

Which platforms support Markdown tables?

GitHub, GitLab, Bitbucket, Notion, Obsidian, Jekyll, Hugo, and most modern Markdown editors.

What if my CSV has many columns?

Markdown tables support any number of columns, though readability may decrease with many columns. Use the Column Picker tool first to select only the columns you need.

Is my file uploaded to a server?

No. All conversion happens entirely in your browser.
