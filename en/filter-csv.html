💡 Tips

  • The header row is always preserved in the output

FAQ

Can I apply multiple conditions at once?

Currently only a single condition is supported. For complex multi-condition filtering, use the Spreadsheet Editor.

How are empty cells handled?

Empty cells are treated as empty strings (""). Use the "Equals" condition with an empty value to extract rows with blank cells.

Is the original file modified?

No. The original file is never changed. Only the filtered result is saved as a new file.
