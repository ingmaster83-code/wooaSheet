💡 Tips

  • Unlike CSV Statistics (simple per-column stats), this tool cross-tabulates by two dimensions — row and column
  • If the value field is text, only the "Count" aggregation is meaningful
  • The "Total" row/column is recalculated from all underlying values, not an average of averages

FAQ

Can I nest multiple columns in rows or columns?

Currently the tool supports a simple pivot with one row field, one column field, and one value field.

What happens if the value column isn't numeric?

Sum, average, max and min only aggregate numeric values. Count works regardless of value type.

Are my files uploaded to the internet?

No. All processing happens entirely in your browser and files are never sent to a server.
