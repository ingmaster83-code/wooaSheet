FAQ

How many files can I merge?

As many as your browser's memory allows. We recommend up to 20MB per file for best performance.

Can I merge files with different column counts?

Yes. Files with fewer columns will have empty values in the remaining columns.

Can I change the order of files?

Files are merged in selection order. To change the order, reselect the files in the desired sequence.
