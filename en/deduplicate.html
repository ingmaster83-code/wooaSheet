FAQ

Which row is kept when duplicates are found?

The first occurrence of a duplicate row is kept. All subsequent duplicates are removed.

Is matching case-sensitive?

Yes, matching is case-sensitive. "Apple" and "apple" are treated as different values.

Is the original file modified?

No. The original file is not changed. The result is downloaded as a new file.
