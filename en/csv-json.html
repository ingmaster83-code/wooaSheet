💡 Tips

  • CSV → JSON: The first row is used as the header and becomes JSON keys
  • JSON → CSV: Only array-of-objects format ([{...}, {...}]) is supported
  • Output JSON is formatted with 2-space indentation for readability

FAQ

Can a single JSON object (not an array) be converted?

Yes — a single object is automatically wrapped in an array and converted to a one-row CSV.

How are nested JSON objects handled?

Nested objects are converted to strings. This tool is optimized for flat key-value arrays.

Is my file sent to a server?

No. All conversion happens entirely in your browser — nothing is uploaded.
