💡 Tips

  • The first row is used as column names. English column names are recommended.
  • Numeric values are inserted without quotes; strings are wrapped in single quotes.
  • INSERT OR IGNORE: Skips rows on duplicate key conflict (SQLite).
  • INSERT OR REPLACE: Overwrites existing rows on duplicate key conflict (SQLite).
  • Single quotes inside values are automatically escaped as ''.

FAQ

How are numbers and strings distinguished?

If a cell value consists only of digits, it's treated as a number. Otherwise it's treated as a string wrapped in single quotes.

Can I process large CSV files?

Processing happens in your browser's memory, so tens of thousands of rows are supported. Very large files depend on your device's performance.

Is my file uploaded to a server?

No. All conversion happens entirely in your browser.
