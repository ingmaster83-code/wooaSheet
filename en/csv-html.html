FAQ

Where can I use the generated HTML?

You can paste the code into any blog, WordPress, Notion, or website with an HTML editor and the table will display immediately.

Can I upload Excel files?

Yes. Upload .xlsx or .xls files and the tool automatically reads the first sheet and converts it.

Is my file uploaded to a server?

No. All conversion happens in your browser. Nothing is sent to any server.

My characters appear garbled — what do I do?

Make sure your CSV file is saved in UTF-8 encoding. You can use the Encoding Converter tool to fix this.
