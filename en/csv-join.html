💡 Tips

  • This is different from CSV Merge (stacking files on top of each other) — this tool combines two files side by side by a common key column
  • Left Join keeps all base rows; Inner Join keeps only rows matched on both sides
  • Large files with tens of thousands of rows are processed in a Web Worker so the page never freezes

FAQ

Can I match files even if the column names differ?

Yes. You can select the matching key column separately for each file, so the files can be matched even if the column names are different.

What happens if the key column has duplicate values?

If the key column in either file has duplicate values, every matching combination is generated as a separate result row, increasing the total row count. The tool checks for duplicates automatically after upload.

Are my files uploaded to the internet?

No. All processing happens entirely in your browser and files are never sent to a server.
