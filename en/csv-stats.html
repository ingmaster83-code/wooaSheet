FAQ

Where can I see the median?

Currently only the mean (average) is provided. Use the MEDIAN function in the Spreadsheet Editor for the median.

Can I save the statistics results?

Use your browser's Print (Ctrl+P) function to save as PDF, or copy the table manually.

Is my file sent to a server?

No. All analysis happens entirely in your browser — nothing is uploaded.
