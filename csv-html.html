CSV → HTML 테이블 변환기

CSV 파일 또는 텍스트를 HTML 테이블 코드로 즉시 변환. 스타일·헤더 설정 후 코드를 복사하세요.

🌐

CSV 파일을 여기에 놓으세요

.csv .xlsx .xls 파일 | 최대 20MB

💡 사용 팁

  • 첫 번째 행이 헤더라면 헤더 사용 옵션을 켜면 <thead>로 분리됩니다
  • 인라인 스타일 포함을 켜면 외부 CSS 없이도 바로 적용됩니다
  • 블로그·노션·워드프레스 등에 HTML 코드를 그대로 붙여넣을 수 있습니다
  • Excel(.xlsx) 파일도 직접 업로드 가능합니다

FAQ

생성된 HTML을 어디에 쓸 수 있나요?

블로그, 워드프레스, 노션, 티스토리 등 HTML 편집이 가능한 곳 어디서나 붙여넣으면 바로 표가 표시됩니다.

Excel 파일도 변환되나요?

네, .xlsx / .xls 파일을 업로드하면 첫 번째 시트를 자동으로 읽어 변환합니다.

파일이 서버에 저장되나요?

아니요. 모든 변환은 브라우저에서만 처리됩니다. 파일은 서버로 전송되지 않습니다.

한글이 깨져 보이면 어떻게 하나요?

CSV 파일의 인코딩을 UTF-8로 변환한 후 업로드해 보세요. 인코딩 변환 도구를 사용하시면 편리합니다.
