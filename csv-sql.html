자주 묻는 질문

생성된 SQL을 바로 데이터베이스에 실행할 수 있나요?

MySQL, PostgreSQL 등 표준 SQL 문법으로 생성됩니다. DB 종류에 따라 일부 수정이 필요할 수 있습니다.

테이블 이름을 직접 설정할 수 있나요?

네. 변환 옵션에서 테이블 이름을 지정할 수 있습니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 개인정보와 민감한 데이터가 외부로 유출될 걱정이 없으며, 기업 보안 환경에서도 안심하고 사용할 수 있습니다.

💡 사용 팁

  • 첫 행이 열 이름(컬럼명)으로 사용됩니다. 영문 컬럼명을 권장합니다.
  • 숫자 값은 따옴표 없이, 문자열 값은 작은따옴표로 감싸집니다.
  • INSERT OR IGNORE: 중복 키 충돌 시 해당 행을 건너뜁니다 (SQLite).
  • INSERT OR REPLACE: 중복 키 충돌 시 기존 행을 덮어씁니다 (SQLite).
  • 값 안의 작은따옴표(')는 자동으로 이스케이프('')됩니다.
