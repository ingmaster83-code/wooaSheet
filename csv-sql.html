CSV → SQL 변환기

CSV 파일을 SQL INSERT문으로 즉시 변환합니다. 테이블명과 INSERT 종류를 설정할 수 있습니다.

📄

CSV 파일을 여기에 놓으세요

.csv .xlsx 파일 | 최대 20MB

SQL 미리보기

💡 사용 팁

  • 첫 행이 열 이름(컬럼명)으로 사용됩니다. 영문 컬럼명을 권장합니다.
  • 숫자 값은 따옴표 없이, 문자열 값은 작은따옴표로 감싸집니다.
  • INSERT OR IGNORE: 중복 키 충돌 시 해당 행을 건너뜁니다 (SQLite).
  • INSERT OR REPLACE: 중복 키 충돌 시 기존 행을 덮어씁니다 (SQLite).
  • 값 안의 작은따옴표(')는 자동으로 이스케이프('')됩니다.

FAQ

숫자와 문자열을 어떻게 구분하나요?

CSV 셀 값이 숫자로만 이루어진 경우 숫자로 처리하고, 그 외에는 문자열(작은따옴표)로 처리합니다.

대용량 CSV도 처리되나요?

브라우저 메모리 내에서 처리되므로 수만 행도 가능하지만, 매우 큰 파일은 브라우저 성능에 따라 다를 수 있습니다.

파일이 서버에 전송되나요?

아니요. 모든 변환은 브라우저에서만 처리됩니다.
