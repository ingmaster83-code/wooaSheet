자주 묻는 질문

CSV의 첫 행이 키로 인식되나요?

네. 첫 행이 헤더로 처리되어 JSON 키가 됩니다.

JSON → CSV 변환 시 배열 형식이어야 하나요?

객체 배열([{...}, {...}]) 형식만 지원합니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

💡 사용 팁

  • CSV → JSON: 첫 행이 헤더로 사용되어 JSON 키가 됩니다
  • JSON → CSV: 객체 배열 ([{...}, {...}]) 형식만 지원합니다
  • 결과 JSON은 들여쓰기 2칸이 적용된 읽기 쉬운 형식입니다
