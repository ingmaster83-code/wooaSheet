💡 사용 팁

  • CSV → JSON: 첫 행이 헤더로 사용되어 JSON 키가 됩니다
  • JSON → CSV: 객체 배열 ([{...}, {...}]) 형식만 지원합니다

FAQ

JSON 배열이 아닌 단일 객체도 변환되나요?

단일 객체는 자동으로 배열로 감싸져 1행짜리 CSV로 변환됩니다.

중첩 JSON은 어떻게 처리되나요?

중첩 객체는 문자열로 변환됩니다. 단순 키-값 구조의 배열에 최적화되어 있습니다.

파일이 서버에 전송되나요?

아니요. 모든 변환은 브라우저에서만 처리됩니다.
