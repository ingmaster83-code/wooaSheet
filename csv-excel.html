자주 묻는 질문

한글이 깨지지 않나요?

UTF-8 BOM 인코딩으로 저장하면 Excel에서 한글이 깨지지 않습니다.

여러 시트를 포함할 수 있나요?

현재는 단일 시트로 변환됩니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

💡 변환 팁

  • CSV → Excel: 한글이 포함된 경우 UTF-8 BOM이 자동 적용됩니다
  • Excel → CSV: 첫 번째 시트만 변환됩니다
  • 수식은 계산된 값으로 변환됩니다

자주 묻는 질문

한글이 깨지지 않나요?

CSV 변환 시 UTF-8 BOM을 자동으로 추가하여 Excel에서 한글이 정상적으로 표시됩니다.

여러 시트가 있는 Excel은요?

Excel → CSV 변환 시 첫 번째 시트만 변환됩니다. 다른 시트는 편집기에서 불러올 수 있습니다.

파일이 서버에 저장되나요?

아니요. 모든 변환은 브라우저에서만 처리됩니다. 파일이 외부로 전송되지 않습니다.
