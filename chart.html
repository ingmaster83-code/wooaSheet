자주 묻는 질문

어떤 차트 유형을 지원하나요?

막대, 선, 파이, 산점도 등 다양한 차트 유형을 지원합니다.

차트를 이미지로 저장할 수 있나요?

네. PNG 또는 SVG 형식으로 저장할 수 있습니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 개인정보와 민감한 데이터가 외부로 유출될 걱정이 없으며, 기업 보안 환경에서도 안심하고 사용할 수 있습니다.

💡 사용 팁

  • 첫 행은 헤더(열 이름)로 인식됩니다
  • 파이/도넛 차트는 숫자 데이터 열을 선택하세요
  • PNG 저장 시 배경이 흰색으로 저장됩니다

자주 묻는 질문

어떤 파일 형식을 지원하나요?

CSV, XLSX, XLS 파일을 지원합니다. 최대 10MB까지 업로드 가능합니다.

차트를 어떻게 저장하나요?

PNG로 저장 버튼을 누르면 고해상도 PNG 이미지로 다운로드됩니다.

데이터가 외부로 전송되나요?

아니요. 모든 처리는 브라우저 내에서만 이루어지며 외부 서버로 전송되지 않습니다.
