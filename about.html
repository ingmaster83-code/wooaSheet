WooaSheet 소개

WooaSheet는 별도 설치나 회원가입 없이 브라우저에서 바로 사용하는 무료 온라인 스프레드시트 도구 모음입니다. 모든 파일 처리는 사용자 기기에서 이루어지며, 서버로 어떠한 데이터도 전송되지 않습니다.

제공 도구

  • 스프레드시트 편집기 — 엑셀 수식 지원 온라인 편집기 (Handsontable + HyperFormula)
  • CSV ↔ Excel 변환 — CSV와 Excel 파일을 빠르게 상호 변환
  • 차트 생성기 — CSV 데이터로 막대·꺾은선·원형·도넛 차트 생성 및 PNG 다운로드
  • 중복 데이터 제거 — CSV/Excel 내 중복 행을 자동으로 찾아 제거
  • CSV 파일 병합 — 여러 CSV 파일을 하나로 합치기
  • CSV 파일 분할 — 대용량 CSV를 일정 행 단위로 분할하여 ZIP 다운로드

개인정보 보호

업로드한 파일은 서버로 전송되지 않으며, 브라우저 메모리에서만 처리됩니다. 페이지를 닫으면 모든 데이터가 완전히 삭제됩니다.
