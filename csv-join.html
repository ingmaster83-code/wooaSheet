CSV 매칭·조인

두 CSV 파일을 공통 열(키) 기준으로 매칭해서 옆으로 결합합니다. 엑셀 VLOOKUP·INDEX-MATCH를 대체합니다.

💡

여러 CSV를 위아래로 이어 붙이려면 → CSV 합치기를 이용하세요.
이 도구는 두 CSV를 공통 열(키) 기준으로 옆으로 결합하는 매칭·조인 전용 도구입니다 (열 구조가 같은 파일을 이어붙이는 것과는 다른 기능입니다).

자주 묻는 질문

컬럼명이 달라도 매칭할 수 있나요?

네. 기준 파일과 조인 파일에서 매칭 기준이 될 열을 각각 따로 선택할 수 있어, 컬럼명이 달라도 매칭됩니다.

키 값이 중복되면 결과가 어떻게 되나요?

기준 파일 또는 조인 파일의 키 값이 중복되면, 매칭되는 모든 조합이 각각 결과 행으로 생성되어 최종 행 수가 늘어날 수 있습니다. 업로드 후 중복 여부를 자동으로 확인해 안내합니다.

파일이 인터넷에 업로드되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 개인정보와 민감한 데이터가 외부로 유출될 걱정이 없으며, 기업 보안 환경에서도 안심하고 사용할 수 있습니다.

💡 사용 팁

  • CSV 합치기(여러 파일을 위아래로 잇기)와는 다른 기능입니다 — 이 도구는 두 파일을 공통 열 기준으로 옆으로 결합합니다
  • Left Join은 기준 파일 행을 모두 유지하고, Inner Join은 양쪽에서 매칭된 행만 남깁니다
  • 수만 행의 대용량 파일도 Web Worker에서 처리되어 화면이 멈추지 않습니다
