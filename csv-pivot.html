자주 묻는 질문

행·열에 여러 컬럼을 중첩할 수 있나요?

현재는 행 1개, 열 1개, 값 1개로 구성된 단순 피벗테이블을 지원합니다.

값 열이 숫자가 아니면 어떻게 되나요?

합계·평균·최대·최소는 숫자만 집계합니다. 개수(count)는 값의 종류와 무관하게 해당 조건의 데이터 행 수를 셉니다.

파일이 인터넷에 업로드되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 개인정보와 민감한 데이터가 외부로 유출될 걱정이 없으며, 기업 보안 환경에서도 안심하고 사용할 수 있습니다.

💡 사용 팁

  • CSV 통계 분석(열별 단순 통계)과 다르게, 이 도구는 행·열 두 기준으로 교차 집계합니다
  • 값 열이 숫자가 아닌 텍스트라면 "개수" 집계만 의미가 있습니다
  • 결과 표의 "총계" 행·열은 선택한 집계 방식으로 전체를 다시 계산한 값입니다 (평균의 평균이 아닙니다)
