CSV 유효성 검사기

CSV 파일의 오류를 자동으로 검출합니다. 열 수 불일치, 빈 행, 중복 행, 빈 헤더 등을 한번에 확인.

📄

CSV 파일을 여기에 놓으세요

.csv .xlsx 파일 | 최대 20MB

✅ 문제를 찾지 못했습니다. CSV가 유효합니다!

    자주 묻는 질문

    유효성 검사 기준을 직접 설정할 수 있나요?

    열 타입(숫자, 날짜, 이메일 등), 필수 여부, 범위 등을 지정할 수 있습니다.

    오류가 있는 행을 별도 파일로 내보낼 수 있나요?

    네. 오류 행만 분리해 CSV로 저장할 수 있습니다.

    파일이 서버에 저장되나요?

    아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

    💡 검사 항목

    • 빈 헤더: 첫 행(헤더)에 빈 열 이름이 있는지 확인
    • 열 수 불일치: 헤더와 데이터 행의 열 수가 다른 경우 감지
    • 완전히 빈 행: 모든 셀이 비어 있는 행 위치 표시
    • 중복 행: 모든 열 값이 동일한 중복 행 탐지
    • 빈 값 통계: 열별 빈 값(NULL) 개수 요약
