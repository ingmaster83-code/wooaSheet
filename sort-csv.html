CSV 정렬

특정 열을 기준으로 CSV 데이터를 오름차순·내림차순으로 정렬합니다.

🔀

CSV 또는 Excel 파일을 여기에 놓으세요

.csv .xlsx 파일 | 최대 50MB

전체 행
열 수

자주 묻는 질문

여러 열 기준으로 정렬할 수 있나요?

네. 첫 번째 기준 열 정렬 후 두 번째 기준으로 추가 정렬이 가능합니다.

숫자와 텍스트를 구분해서 정렬하나요?

네. 숫자형 데이터는 숫자 크기 기준으로, 텍스트는 사전 순으로 정렬됩니다.

파일이 서버에 저장되나요?

아니요. 모든 처리는 브라우저 안에서만 이루어지며 파일은 서버로 전송되지 않습니다. 완전히 안전합니다.

💡 사용 팁

  • 날짜는 YYYY-MM-DD 형식이면 텍스트 정렬로도 정확하게 정렬됩니다
  • 숫자가 포함된 열은 "숫자" 타입을 선택하면 1, 2, 10 순서로 정렬됩니다
  • 빈 셀은 정렬 시 마지막으로 배치됩니다
